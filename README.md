This test example covers:
- Logging into the web version of Base.
- Creating a new Lead
- Checking it's status
- Changing default status to all leads.
- Checking that status change affected created lead.

Tests are created using:

[Selenium WebdriverJS](https://github.com/SeleniumHQ/selenium/tree/master/javascript)

[Mocha](https://mochajs.org/)

[Chai](http://chaijs.com/)
as assertion library.

To run tests all you need to do is:

```
git clone
npm install
npm test
```

`Npm install` will handle all of the dependencies, and `npm test` command will handle mocha test runner.
